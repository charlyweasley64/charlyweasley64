Instalar macOS Catalina en VMware con NVDA y VoiceOver

Este tutorial te guía paso a paso para instalar macOS Catalina en VMware Workstation en Windows, usar NVDA para controlar la máquina virtual y configurar VoiceOver dentro de macOS para accesibilidad.

  • ISO oficial macOS Catalina (desde Internet Archive - método alternativo)
  • VMware Workstation Player (Directorio de jake groxgan)
  • NVDA - lector de pantalla para Windows
  • Guía oficial de VoiceOver en macOS (Apple)
  • VMware Unlocker para habilitar macOS en VMware

Nota: Descargar una ISO de macOS directamente no está disponible oficialmente, pero existen métodos legítimos para crearla desde un Mac o mediante herramientas de terceros confiables.

  1. Instala VMware Workstation Pro desde el instalador descargado.
  2. Descarga y ejecuta el VMware Unlocker para habilitar la opción de macOS en VMware. Sigue las instrucciones del repositorio para parchear VMware.
  3. Reinicia tu PC para aplicar los cambios.
  1. Abre VMware y elige Crear una nueva máquina virtual.
  2. Selecciona Instalar desde imagen ISO y busca la ISO de macOS Catalina.
  3. Selecciona Apple Mac OS X y la versión macOS 10.15.
  4. Configura la memoria RAM (recomendado mínimo 4 GB) y disco duro (mínimo 40 GB, recomendado SSD para mejor rendimiento).
  5. Finaliza la creación de la VM.
  1. Entra a la ruta donde creaste la VM, busca el archivo nombre_de_la_máquina.vmx, dale clic derecho o tecla aplicaciones (Shift + F10) y elige Editar con bloc de notas.
  2. Agrega la siguiente línea al final del archivo y guarda:
    smc.version = "0"
  3. Inicia la VM recién creada.
  4. Espera a que aparezca el logotipo de Apple y luego el selector de idioma.
  5. Pulsa Cmd + F5 para activar VoiceOver y usa las flechas para buscar "Español".
  6. Presiona Tab hasta el botón Continuar y actívalo.
  7. Usa las flechas para seleccionar "Utilidad de Discos" y presiona Tab hasta el botón Continuar, luego actívalo.
  8. En Utilidad de Discos, selecciona Virtual SATA Hard Drive Media, presiona Tab hasta Borrar y confirma.
  9. Asigna un nombre al disco, presiona Tab hasta Borrar y confirma con Espacio, luego presiona Cmd + Q para salir.
  10. Busca con flechas la opción "Instalar macOS Catalina", presiona Tab hasta Continuar y sigue las instrucciones en pantalla.
  11. Cuando llegues a la configuración inicial, activa VoiceOver con Cmd + F5 para usarlo en la configuración.

VoiceOver es el lector de pantalla integrado en macOS. Puedes activarlo con Cmd + F5.

  • Permite navegar, leer textos, describir botones y más dentro de macOS.
  • Prueba comandos básicos como Ctrl + Option + Flecha Derecha para moverte entre elementos.
  • Consulta la guía oficial de VoiceOver para más detalles.
  • Paciencia: la instalación puede tardar y pedir reinicios.
  • Configura el teclado para que las teclas especiales funcionen bien entre Windows y macOS virtualizado.
  • Consulta la comunidad de Apple y foros de VMware para problemas comunes.
  • Guarda tu progreso y usa snapshots en VMware para no perder tu VM configurada.
